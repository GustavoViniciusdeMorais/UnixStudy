# Alias commands
```bash
alias # list the alias options

tail -n 3 /root/.bashrc

echo "alias tlp='netstat -tlp'" >> /root/.bashrc

tail -n 3 /root/.bashrc

source /root/.bashrc

tlp

alias
```
### Usefulls
```bash
alias mybranch='git branch --show-current'
alias myhtml='cd /var/www/html'
alias myowner='chown -R gustavo:gustavo ./'
```
